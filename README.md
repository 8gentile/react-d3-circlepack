A React/d3 implementation of a circle pack diagram.

### Importing library

You can import the Component with either

```javascript
import ReactD3CirclePack from 'react-d3-circlepack'
```
or

```javascript
const ReactD3CirclePack = require('react-d3-circlepack')
```
